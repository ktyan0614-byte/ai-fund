# AI 虛擬投資公司

一個全自動的虛擬投資系統:由 AI 依照真實台股股價每週自動決策買賣,
並產出中文週報。**不涉及任何真實金錢。**

系統同時運行**三個各 10 萬元(虛擬)的帳戶**,做策略的平行對照實驗:

| 帳戶 | 策略 | 一句話說明 | 19年回測:年化/回檔/夏普 |
|---|---|---|---|
| 一:純動能 | 動能 Top5 + 大盤濾網 | 只看「股價最近漲多強」 | +21.7% / -40% / 1.00 |
| 二:動能+營收 | 動能與月營收年增率各半 | 一半看股價,一半看「生意成長多快」 | +22.6% / -34% / 1.08 |
| 三:動能+毛利 | 動能與毛利率年變化各半 | 一半看股價,一半看「生意有沒有變好賺」 | +24.8% / -34% / 1.15 |
| 四:美股ETF | SPY 50%/QQQ 30%/VIG 20% 被動持有 | 不選股不擇時,偏離配置>5pp才再平衡 | +12.7% / -49% / 0.65(台幣) |

帳戶四是對照組中的對照組:美股被動部位,不受台股濾網管控,
用來檢驗「主動策略到底值不值得做」與提供地理分散。

台股上市公司依法每月 10 日前公佈月營收(全世界最即時的基本面數據),
季報則揭露毛利率——帳戶二、三分別利用這兩層基本面。
回測排名只是「參賽資格」;誰能留下,由未來幾個月的樣本外表現決定。

## 系統怎麼運作(白話版)

每週五台股收盤後,系統會自動:

1. **抓最新股價** — 26 檔台股各產業大型權值股 + 0050(大盤指標)
2. **看大盤臉色(濾網)** — 0050 跌破 120 日均線 = 市場轉弱,全部賣掉抱現金;
   站上均線才進場
3. **挑最強的股票(動能)** — 把 26 檔股票依「近 60 個交易日漲幅」排名,
   買進前 5 名、每檔各占 1/5 資金
4. **模擬成交** — 以收盤價買賣,計入真實的手續費(0.0855%)與證交稅(0.3%)
5. **產出週報** — `reports/日期.md`,含四大內容:
   操作內容、決策依據、市場方向、產業近況

## 為什麼用這套策略?

用 2020–2026 年真實股價回測比較(皆含交易成本):

| 策略 | 年化報酬 | 最大回檔 | 夏普值 | 2022 空頭年 |
|---|---|---|---|---|
| 買進持有 0050(基準) | +31.2% | -33.8% | 1.39 | -21.2% |
| 動能 Top5(無濾網) | +55.7% | -46.2% | 1.55 | -34.7% |
| **動能 Top5 + 濾網(帳戶一)** | **+59.3%** | -38.5% | **1.71** | -20.8% |
| **動能+營收混合 + 濾網(帳戶二)** | +54.0% | **-34.3%** | 1.70 | **-7.2%** |
| 動能 Top5 限營收成長股 + 濾網 | +51.4% | -36.6% | 1.62 | -15.9% |
| 0050 均線進出 | +24.6% | -30.5% | 1.33 | -20.3% |

純動能報酬最高;動能+營收報酬略低但回檔更淺、空頭年明顯抗跌、夏普值打平——
風格不同而非優劣,所以兩者平行實驗。

### 成交價敏感度測試(回測 vs 實際下單的差距)

同一套「動能 + 濾網」策略,換不同的成交假設:

| 成交假設 | 年化報酬 | 最大回檔 | 夏普值 |
|---|---|---|---|
| 週五收盤價成交(回測預設,最樂觀) | +59.3% | -38.5% | 1.71 |
| 下週一**開盤價**成交(實際可執行) | +56.1% | -39.5% | 1.66 |
| 買在隔日**最高點**、賣在**最低點**(FOMO 追高殺低) | +18.2% | -64.1% | 0.72 |

結論:策略本身對「隔天才成交」不敏感(年化只差約 3%),
但**追高殺低會吃掉三分之二的報酬、回檔擴大近一倍**。
所以週報附有「決策卡」:買單一律掛限價(上限 = 收盤價 +2%),買不到就放棄。

注意:回測期間台股大多頭,未來實際表現通常會比回測遜色。

### 長期回測(2007–2026,約 19 年,含 2008 金融海嘯)

6 年資料不足以評估策略(幾乎都是多頭),拉長到 19 年、用加權指數當濾網:

| 策略 | 年化報酬 | 最大回檔 | 夏普值 | 2008 海嘯年 |
|---|---|---|---|---|
| 持有大盤指數(不含息) | +9.5% | -58.0% | 0.59 | -46% |
| 動能無濾網 | +22.6% | -65.8% | 0.92 | -53% |
| 動能+濾網(帳戶一) | +21.7% | -40.0% | 1.00 | **-11%** |
| 動能+營收+濾網(帳戶二) | +23.0% | **-33.7%** | **1.09** | **-9%** |

三個重要發現:
1. **近 6 年的 +59% 年化被多頭灌水了 2.5 倍以上**,19 年下來動能策略年化約 +22%,
   合理的長期預期應該再打折(見下方倖存者偏誤)
2. **濾網在 2008 是生死之別**:無濾網 -53%,有濾網 -11%
3. **動能有連續好幾年的低潮期**(2012-13、2018、2024 落後大盤),
   多數人就是在低潮期棄守,然後錯過下一波

誠實的限制:投資範圍是「2026 年的權值股」,2007 年的你不可能未卜先知選中
這些倖存的贏家(倖存者偏誤),所以長期絕對報酬仍被高估;
另外大盤指數不含股息,實際持有 0050 的年化約 +13% 而非 +9.5%。
把這些都算進去,策略對大盤的真實優勢大約是每年幾個百分點,不是回測表面的數字。

## 檔案結構

```
config.py            投資範圍、策略參數、費率設定
data.py              股價下載(yfinance)
fundamentals.py      月營收下載與年增率計算(FinMind API)
strategy.py          策略邏輯(動能排名、動能+營收綜合排名、大盤濾網)
backtest.py          回測程式(比較六種策略 + 成交價敏感度)
run_weekly.py        每週執行主程式(雙帳戶決策 + 交易 + 週報)
portfolio/portfolio.json          帳戶一:純動能
portfolio/portfolio_hybrid.json   帳戶二:動能+營收
portfolio/trades*.csv             各帳戶交易紀錄
reports/YYYY-MM-DD.md             每週報告(兩帳戶合併)
.github/workflows/weekly.yml      GitHub Actions 自動排程
```

## 本機執行

```bash
pip install -r requirements.txt
python run_weekly.py     # 執行一次每週決策 + 產出週報
python backtest.py       # 重跑回測
```

## 放上 GitHub 自動運行

1. 建立新 repo,把整個資料夾推上去
2. 不用設任何 Secret(股價是公開資料,commit 用內建的 GITHUB_TOKEN)
3. 到 repo 的 Settings → Actions → General,把 Workflow permissions
   設為 **Read and write permissions**
4. 之後每週五 15:00(台北時間)自動執行,週報與帳戶變動會自動 commit 回 repo;
   也可以到 Actions 頁面手動按 Run workflow 測試
